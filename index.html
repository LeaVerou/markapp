Building apps by authoring HTML

1 in 2 HTML & CSS authors are not very comfortable with authoring JS.

Every time you require JS code to initialise or configure your UI library you are excluding them. Even developers often find the declarative nature of HTML quicker to work with. The goal of this site is to collect and promote JS libraries that can be used without writing any JS.
